C ChannelView API Documentation & Setup Guide

Quick Start

ChannelView is a one-way async video interview platform built for insurance agencies. Use the REST API to integrate with your ATS, CRM, or internal tools.

v1.0 All endpoints require authentication via API key or session cookie.

Authentication

Generate an API key from Settings > API in your dashboard. Include it in every request:

curl -H "X-API-Key: cv_your_key_here" https://your-domain.com/api/v1/interviews

REST API Endpoints

GET /api/v1/interviews
List all interviews in your account.
GET /api/v1/interviews/:id/candidates
List all candidates for a specific interview.
GET /api/v1/candidates/:id
Get detailed candidate info including responses and scores.
POST /api/v1/candidates
Create and invite a new candidate. Body: {"interview_id", "first_name", "last_name", "email"}

Webhooks & Events

Configure webhooks from Settings > Integrations or connect via Zapier. Events are emitted in real-time:

candidate.invited — Candidate was sent an interview invitation

candidate.started — Candidate opened and started their interview

candidate.completed — Candidate submitted all responses

candidate.scored — AI or manual scoring was applied

candidate.hired / candidate.rejected — Status changed

interview.created / interview.closed — Interview lifecycle

Webhook payloads include a JSON body with event, timestamp, and data fields. If a secret is configured, an X-Webhook-Signature header is included (HMAC-SHA256).

Zapier Integration

Paste your Zapier webhook URL in Integrations to connect ChannelView to 5,000+ apps. Events are automatically forwarded in Zapier-compatible format.

FMO Setup Guide

For FMO Administrators

As an FMO admin, you have access to the Admin Panel with platform-wide statistics, account listings, and the ability to manage agencies under your umbrella.

Onboarding new agencies: Each agency goes through a 4-step wizard (profile, branding, first interview, completion). They can customize their brand colors, logo, and candidate-facing name.

White-label: Agencies can enable white-label mode to show their own branding to candidates instead of the ChannelView brand.

Billing: ChannelView Pro is $99/month per agency. Agencies manage their own subscriptions via Stripe.

Agency Setup Checklist

1. Complete the 4-step onboarding wizard

2. Configure SMTP or SendGrid for email delivery

3. Upload your agency logo and set brand colors

4. Create your first interview template with questions

5. Invite candidates individually or via CSV bulk import

6. Set up automation rules (auto-score, advance/reject thresholds)

7. Configure webhook integrations or Zapier connection

8. Review compliance settings and data retention policies
